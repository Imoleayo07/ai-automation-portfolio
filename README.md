## Projects

### 1. Mama Tee's Kitchen - AI Voice Concierge
**Tools:** Vapi, n8n, Google Sheets, Telegram, Groq (Llama 3.3 70b)

A fully automated voice assistant that answers inbound calls for a 
fast-casual Nigerian restaurant in Abuja. The assistant handles food 
orders, table reservations, and general enquiries in natural Nigerian 
English. All call data is logged to Google Sheets and real-time 
Telegram notifications are sent for every order, reservation, and 
escalation. Built with a custom Vapi tool function (submitCallSummary) 
that fires structured data to n8n at the end of every call.

**Key features:**
- Answers calls 24/7 without human intervention
- Collects full order details including items, sizes, proteins, and delivery address
- Calculates total estimates using exact menu pricing
- Routes call types through a Switch node to separate Google Sheets branches
- Sends Telegram alerts to the restaurant owner in real time
