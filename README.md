# AI Automation Portfolio

A collection of AI-powered automation workflows and engineering projects built with n8n, Groq, and various data tools.

## Projects

### 1. LinkedIn Content Automation
**Tools:** n8n, Groq (Llama 3.3 70b), Google Sheets, Gmail, Make.com, LinkedIn API  
Automated LinkedIn post generation and publishing on a Mon/Wed/Fri schedule. Groq generates content from a Google Sheets content plan, Gmail handles approval, and Make.com triggers the final LinkedIn publish.

### 2. AI News Agent
**Tools:** n8n, Groq (Llama 3.3 70b), SerpAPI, Telegram, Window Buffer Memory  
An agent that searches for recent news, summarizes it using an LLM, and delivers formatted updates to a Telegram channel. Resolved tool-calling model compatibility and Telegram output sanitization during build.

### 3. Gym Membership Management System
**Tools:** n8n, Airtable, Gmail  
A 3-part automation that manages gym member records, sends renewal reminders, and handles subscription carry-over day logic via a JavaScript Code node.

### 4. RAG Pipeline for Think Stats
**Tools:** n8n, Google Drive, HuggingFace Embeddings, Supabase, Groq  
A retrieval-augmented generation pipeline that indexes a PDF textbook and answers questions from it. Built as part of an AI engineering assignment.

### 5. Coach Max - Motivational Math Tutor
**Tools:** Claude API, HTML/CSS/JS  
A browser-based interactive math tutor built through system prompt engineering. Designed to be encouraging and step-by-step in its explanations.

## Tech Stack
- **Automation:** n8n Cloud
- **LLM:** Groq API (Llama 3.3 70b-versatile)
- **Databases:** Airtable, Supabase, Google Sheets
- **Notifications:** Gmail, Telegram
- **Publishing:** LinkedIn API via Make.com
